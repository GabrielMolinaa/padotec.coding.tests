Como Rodar com Docker
Pré-requisitos:

    Docker instalado

    Git instalado

Passos:

# Clone o repositório
git clone https://github.com/GabrielMolinaa/padotec.coding.tests.git
cd repositorio_clonado

# Suba os containers da API e do MongoDB
docker-compose up --build

A API estará disponível em:
http://localhost:3000
